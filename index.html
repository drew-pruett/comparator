Comparator Project Page

    The Comparator is a simple web app designed to predict irreversible decompensation in virtual patients exposed to hemorrhage. The underlying engine is a support vector machine-desinged classifier function that takes three consecutive measurements each of systolic and diastolic blood pressure, respiration rate, and heart rate, and outputs a flag that warns of imminent irreversible decompensation.

    This function was designed and tested on virtual patients constructe from HumMod, an integrative whole body physiological simulator. As such, it is not intended for use on humans.

    The comparator is based on Guytonian physiology. It makes the assumption that hemorrhage is pain-free, which avoids complications with vagal interactions with the nociceptor system.

    The comparator was built by W. Andrew Pruett and Leland Husband, with advisory input from Robert Hester in June 2014. Any questions should be addressed to W. Pruett.
